## Scalars

A scalar is a single numerical value that represents a quantity without any direction.

- Person's age = 40 years ✅
- Temperature = 25°C ✅
- Mass = 70 kg ✅
- Speed = 45 km/h ✅ (Speed is a scalar because it has no direction.)
- Distance = 10 km ✅

## Vectors

A vector is a quantity that has both magnitude and direction.

- A car moving 45 km/h towards the east ✅
- Wind blowing 20 km/h towards the north ✅
- A force of 10 N upward ✅

Example:

```bash
Car
--------->

45 km/h East
```

- Magnitude = 45 km/h
- Direction = East

A vector can be represented using coordinates.
Example:

```bash
v = (3, 2)
This means:

Move 3 units along the X-axis.
Move 2 units along the Y-axis.

Y
↑
|
|       • (3,2)
|
|
+----------------→ X
```

## Vector Magnitude

The magnitude of a vector is simply its length.
Imagine a vector starts at the origin (0, 0) and ends at (3, 4).

```bash
Y
↑
|
|          • (3,4)
|        /
|      /
|    /
|  /
|/
+----------------------→ X
(0,0)
```

The line from (0,0) to (3,4) is the vector.
The length of this line is called the vector's magnitude.

### Formula

For a 2D vector:

```bash
v=(x,y)
The magnitude is:


|v| = √(x² + y²) or |v| = sqrt(x^2 + y^2)

Example 1

|v| = sqrt(3^2 + 4^2)
     = sqrt(9 + 16)
     = sqrt(25)
     = 5

Example 2

v = (-5, 12)

|v| = sqrt((-5)^2 + 12^2)
     = sqrt(25 + 144)
     = sqrt(169)
     = 13
​
```

3D Vector

```bash
v = (x, y, z)

Magnitude Formula

|v| = sqrt(x^2 + y^2 + z^2)

Example

v = (2, 3, 6)

|v| = sqrt(2^2 + 3^2 + 6^2)
     = sqrt(4 + 9 + 36)
     = sqrt(49)
     = 7
```

## Vector Addition

Vector addition is the process of combining two vectors to produce a new vector called the resultant vector.
Unlike scalars, vectors have both magnitude and direction, so we cannot simply add their magnitudes.

### Tip-to-Tail Method

To add two vectors:

Draw the first vector.
Place the tail of the second vector at the tip (head) of the first vector.
Draw a new vector from the tail of the first vector to the tip of the second vector.
This new vector is called the resultant vector.

The vector addition is written as:

```bash
R = U + V
where:

U = First vector
V = Second vector
R = Resultant vector

Suppose we have two vectors:
Vector U has a magnitude of 3 units.
Vector V has a magnitude of 2 units.

We cannot simply calculate

3 + 2 = 5

because vectors also have a direction.

We must first arrange them using the Tip-to-Tail Method.

Step 1

U

O--------->



Step 2 (Move V so its tail touches U's tip)

O--------->--------->

     U          V

Resultant Vector
O----------------------------->

             R = U + V
```

learning for Machine Learning, I actually wouldn't spend too much time on graphical vector addition. In ML, vector addition is almost always performed component-wise:

U = (2, 3)
V = (4, 1)

U + V = (2 + 4, 3 + 1)
= (6, 4)

This is the method you'll use in programming (NumPy, TensorFlow, PyTorch) and is much more important than drawing arrows.

## Vector Subtraction

Vector subtraction is the process of finding the difference between two vectors.

Like vector addition, subtraction considers both magnitude and direction.

Instead of subtracting magnitudes directly, we subtract the corresponding components of each vector.

Formula
R = U - V

where:

U = First vector
V = Second vector
R = Resultant vector

Component-wise Subtraction

Suppose

U = (5, 7)
V = (2, 3)

Then

U - V
= (5 - 2, 7 - 3)
= (3, 4)

For Machine Learning, remember this rule:

```bash
(a, b) - (c, d)
=
(a - c, b - d)
```

That's how you'll perform vector subtraction in libraries like NumPy, TensorFlow, and PyTorch.

## Scalar Multiplication

- Scalar multiplication is the process of multiplying every component of a vector by a scalar (a single number).
- A scalar changes the magnitude of a vector and may also change its direction.

Formula
k × V

where:

k = Scalar
V = Vector

Component-wise Multiplication

```bash
Suppose

V = (2, 3)
k = 4

Then

4 × (2, 3)
= (4 × 2, 4 × 3)
= (8, 12)

Another Example

V = (-1, 5)
k = 3

3 × (-1, 5)
= (-3, 15)
```

### Negative Scalar

- If the scalar is negative, the vector's direction is reversed.

```bash
Example:

V = (2, 3)

-2 × V
= -2 × (2, 3)
= (-4, -6)
```

- The vector now points in the opposite direction.

### Effect of the Scalar

| Scalar      | Effect                                       |
| ----------- | -------------------------------------------- |
| `k > 1`     | Increases the vector's length                |
| `0 < k < 1` | Decreases the vector's length                |
| `k = 0`     | Produces the zero vector `(0, 0)`            |
| `k < 0`     | Reverses the direction and scales the length |

Examples

```bash
2 × (3, 4)
= (6, 8)

0.5 × (8, 6)
= (4, 3)

0 × (5, 9)
= (0, 0)
```

### Why is Scalar Multiplication Important in Machine Learning?

- Scalar multiplication is used in almost every ML algorithm.
- Examples:
  - Scaling feature vectors.
  - Updating model weights during Gradient Descent.
  - Adjusting the learning rate.
  - Normalization and optimization algorithms.
- For example, a weight update in Gradient Descent looks like:

new_weights = old_weights - learning_rate × gradient

Notice that learning_rate is a scalar, while gradient is a vector. We multiply the scalar by every component of the gradient before updating the weights.

# ML-AI-Roadmap

## Timeline (Realistic)

- Python: 2–3 weeks
- Math: 1 month
- Data Analysis: 2 weeks
- Machine Learning: 2 months
- Deep Learning: 2 months
- Generative AI & LLMs: 2 months
- MLOps & Deployment: 1 month
- Projects (ongoing): 2–3 months

## Phase 1 Python (2-3 weeks)

Learn only the parts required for AI.

### Topics:

- Variables
- Functions
- Loops
- OOP
- Exception handling
- File handling
- Modules
- Virtual environments
- pip
- Type hints

### Then learn AI libraries:

- NumPy
- Pandas
- Matplotlib
- Seaborn (for visualization)
- Jupyter Notebook

### Practice:

- Load CSV
- Clean data
- Filter data
- Group data
- Plot graphs

## Phase 2: Mathematics (3-5 weeks)

### Linear Algebra

Very important.
Topics:

- Scalars
- Vectors
- Vector magnitude
- Dot product
- Matrix
- Matrix multiplication
- Matrix transpose
- Identity matrix
- Inverse matrix
- Eigenvalues (basic intuition)

### Probability

Topics:

- Probability basics
- Conditional probability
- Bayes theorem
- Normal distribution
- Mean
- Median
- Variance
- Standard deviation

### Calculus

Only enough for ML.
Topics:

- Function
- Slope
- Derivative
- Gradient
- Chain rule
- Partial derivatives

## Phase 3: Data Analysis (2 weeks)

### Learn:

- Missing values
- Outliers
- Encoding
- Scaling
- Feature engineering
- Data visualization
- Correlation

### Using:

- Pandas
- NumPy
- Matplotlib
- Seaborn

## Phase 4: Machine Learning

### Supervised Learning

- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM

### Unsupervised Learning

- K-Means
- Hierarchical clustering
- PCA

### Understand:

- Train/Test split
- Cross validation
- Overfitting
- Underfitting
- Bias
- Variance

### Evaluation metrics:

- Regression
  - MAE
  - MSE
  - RMSE
  - R²
- Classification
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - ROC-AUC

Library:

- Scikit-learn

## Phase 5: Deep Learning

Now start neural networks.

### Topics:

- Perceptron
- Activation functions
- Feed Forward Network
- Backpropagation
- Gradient Descent
- Optimizers
- Batch normalization
- Dropout

Frameworks:

- Choose PyTorch first.
- Later: TensorFlow/Keras

Today, most research and many modern AI tutorials use PyTorch. (July 2, 2026)

This is the most important phase.

Build projects like:

- Spam detection
- Customer churn prediction
- Movie recommendation
- Resume screening AI

- AI customer support chatbot

- Document OCR pipeline

Projects teach far more than courses.
